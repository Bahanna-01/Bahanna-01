Glossary

This glossary explains key HTML terms used in my website and in my learning journey:

HTML
HyperText Markup Language — the standard language for creating web pages.
CSS
Cascading Style Sheets — used to style and layout HTML elements.
<img>
Tag used to display images on a webpage. Requires a src and alt attribute.
src
Specifies the path to the image file to be displayed.
alt
Provides alternative text for an image, useful for accessibility and when the image can't load.
<a>
Anchor tag used to create hyperlinks to other pages or websites.
href
Specifies the destination URL of a link.
<em>
Emphasizes text, usually displayed in italics.
<ul>
Defines an unordered list (bulleted list).
<li>
Defines a list item inside a list (ordered or unordered).
